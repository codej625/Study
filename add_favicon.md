# 파비콘을 설치해보자!

1. 밑에 사이트로 이동
```
https://www.favicon-generator.org/
```

2. 파비콘을 만들 이미지를 입력시키면 알아서 변환이 된다.

3. 변환 된 이미지를 다운받고 사용할 경로에 넣는다.

4. 사이트에서 생성해준 태그를 head단에 추가시킨다.
ex)
```
<link  rel= "apple-touch-icon"  크기= "57x57"  href= "/apple-icon-57x57.png" > 
<link  rel= "apple-touch-icon"  크기= "60x60"  href= "/apple -icon-60x60.png" > 
<link  rel= "apple-touch-icon"  크기= "72x72"  href= "/apple-icon-72x72.png" > 
<link  rel= "apple-touch-icon"  크기= "76x76"  href= "/apple-icon-76x76.png" > 
<link  rel= "apple-touch-icon"  크기 = "114x114"  href= "/apple-icon-114x114.png"> 
<link  rel= "apple-touch-icon"  크기= "120x120"  href= "/apple-icon-120x120.png" > 
<link  rel= "apple-touch-icon"  크기= "144x144"  href= "/ apple-icon-144x144.png" > 
<link  rel= "apple-touch-icon"  크기= "152x152"  href= "/apple-icon-152x152.png" > 
<link  rel= "apple-touch-icon"  크기 = "180x180"  href= "/apple-icon-180x180.png" > 
<링크  rel= "아이콘"  유형= "이미지/png"  크기= "192x192"  href= "/android-icon-192x192.png" > 
<링크  rel= "아이콘"  type= "이미지/png"  크기= "32x32"  href= "/favicon-32x32.png" > 
<링크  rel= "아이콘"  유형= "이미지/png"  크기= "96x96"  href= "/favicon-96x96.png" > 
<link  rel= "아이콘"  유형= "이미지/png"  크기= "16x16"  href= "/favicon-16x16.png" > 
<link  rel= "매니페스트"  href= "/manifest.json" > 
<메타  이름="msapplication-TileColor"  content= "#ffffff" >

```
